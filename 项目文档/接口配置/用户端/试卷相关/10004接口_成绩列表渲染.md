# 成绩排行页接口

## 1.接口描述

接口请求域名:in/toScoreList
试卷管理员端渲染请求
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| examId | 是 | Number | examId |
| groupId | 是 | Number | 组Id |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| scoreList | Array | 成绩列表 |
| status | Number | 状态变量值 |
| msg | String | 状态变量值描述 |

## 4.示例

### 输入

{
    "examId":10086,
    "groupId":"ASX123"
}

### 输出

{   
    "scoreList":[
        {
            
        }
    ]
    "status":1,
    "msg":"列表渲染成功"
}
