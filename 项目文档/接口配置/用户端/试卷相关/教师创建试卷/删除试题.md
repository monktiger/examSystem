# 删除题目

## 1.接口描述

接口请求域名:exam/deleteQuestion
删除试卷中的一个题目
请求方式:get

## 2.输入参数

|  参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| examId | 是 | Number | 试卷id |
| id | 是 | Number | 试题id |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| status | Number | 状态值(0,1) |
| msg | string | 状态描述 |

## 4.用例

### 输入

{
    "examId":1231231,
    "id":2
}

### 输出

{
    "status":1,
    "msg":"删除成功"
}
