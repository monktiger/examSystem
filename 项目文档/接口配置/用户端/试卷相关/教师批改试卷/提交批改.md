# 提交评分

## 1.接口描述

接口请求域名:judge/score
提交某个主观题的分数
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| copyId | 是 | Number | 学生答卷序列号 |
| id | 是 | Number | 该题序号 |
| score | 是 | Number | 该题分数 |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| status | number | 状态值,1成功,0失败 |
| msg | string | 状态描述 |

## 4.用例

### 输入

{
    "copyId":100021,
    "id":3,
    "score":"24"
}

### 输出

{
    "status":1,
    "msg":"分数提交成功"
}
