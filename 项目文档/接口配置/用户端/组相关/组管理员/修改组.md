# 修改组

## 1.接口描述

接口请求域名:group/nodify
修改组
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| name | 是 | String | 修改后组名 |
| groupId | 是 | String | 组Id |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| status | int | 状态值,1成功,0失败 |
| msg | string | 状态描述 |

## 4.用例

### 输入

{
    "name":"东北师范大学第一组",
    "groupId":"ASXERV",
}

### 输出

{
    "status":1,
    "msg":"组修改成功"
}
