# 列出组

## 1.接口描述

接口请求域名:group/getList
获取组
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| openId | 是 | String | 用户id |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| groupList | Arrary | 组列表 |
| status | int | 状态值,1成功,0失败 |
| msg | string | 状态描述 |

## 4.用例

### 输入

{
    "openId":"AXasdadsdadaDV64aaa"
}

### 输出

{
    "groupList":[
        {
            "name":"高三数学特训",
            "group_id":"XHZE46",
        }
    ]
    "status":1,
    "msg":"组获取成功"
}
