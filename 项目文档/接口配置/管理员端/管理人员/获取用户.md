# 获取用户接口

## 1.接口描述

接口请求域名:userAdmin/getUserList
获取用户接口
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| open_id | 是 | String | 用户标识id |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| userList | Arrary | 用户列 |
| status | int | 状态值,1成功,0失败 |
| msg | string | 状态描述 |
