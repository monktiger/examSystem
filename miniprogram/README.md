## 项目理念（详细参看概要分析）

基于微信小程序的特点，我们的小程序应做到即用即走，即启即用，因此在该系统中我们将采用另外一种体系来组合用户。

## 用户群体

我们扩充我们的用户，不仅仅限制于学生，而转变为社会上的多数群体。因为在很多企业公司和产业中定期的考核培训也是必须的。

我们是基于group的概念来组合相应的用户群体。
